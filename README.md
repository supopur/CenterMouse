# CenterMouse
This code centers the mouse
## Requirements
- Python
- Pip
## How to install
1. Install python
2. Open the file install.bat
## How to run
1. Open the file start.bat
## Controls
- q to center the mouse
## How to stop
- Open the command line window where the script is running and press Ctrl + C
